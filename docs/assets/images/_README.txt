KLEOS — assets/images/ folder
=============================
Drop your real poster/mockup PNGs here. The site is built to lazy-load them.

For each case study, name the image to match its page, e.g.:
  bankai.png, aizen.png, penelope.png, kleos-brand.png,
  chainsaw-man.png, apple-concept.png

Then in that case study file, find the placeholder:
  <div class="cs-image"><span>TITLE</span></div>
and replace it with:
  <img src="../assets/images/bankai.png" loading="lazy" alt="BANKAI" />

ALSO NEEDED:
  og-image.png  — 1200x630px, KLEOS wordmark gold on black.
                  This is the preview image shown when the site is shared
                  on WhatsApp / Instagram / Twitter etc.

COMPRESS before uploading (keeps the site fast):
  Use https://tinypng.com — drag the PNG in, download the smaller version.
