KLEOS — sections/ folder
========================
The main page (../index.html) currently has every section written INLINE,
so the site works on GitHub Pages with zero build tools (the simplest setup
for a beginner).

These section files are provided as clean, standalone copies you can edit in
isolation and paste back into index.html when you want to change one section
without scrolling through the whole file. Each file matches the matching
<section> block in index.html.

If you later move to a build tool that supports HTML includes, you can load
these directly. For now: edit here, copy the block into index.html.
